# ETH Wallet bip39 mnemonics Brute Forcer

- Easy to use
- Batteries included

## Usage:
- Install [Node.js](https://nodejs.org/en/download/)
- Download this repository
- Open terminal in the folder where you downloaded this repository
- Run `npm install`
- Run `node index.js`
